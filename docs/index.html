FunctionGPIO

Browser will ask you to select a COM port.
Flashing takes ~30 seconds.

ESP32-S3 — Manual Bootloader Mode

If the flash fails, put the board into bootloader mode manually:

  1. Hold the BOOT button
  2. Press and release RESET
  3. Release BOOT
  4. Click Connect & Install again
Firmware is auto-built from github.com/s1alknau/Nematostella-time-series on every push to the main branch.
